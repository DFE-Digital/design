{% markdown %} A new version of DfE Frontend version 2.1.0, will be made available in March. This new version will include a vertical navigation component and several important fixes to the [header](/design-system/components/header) component. ## New vertical navigation component The new vertical navigation component provides a streamlined solution for creating vertical navigation menus that are both user-friendly and fully accessible. The vertical navigation is designed to work seamlessly across DfE services, enabling teams to create clear and consistent navigation structures that improve the user experience, particularly on mobile and tablet devices. ## Header component fixes In addition to the new vertical navigation, version 2.1.0 also includes important fixes to the header component. These updates address several minor issues reported by users, enhancing the responsiveness and accessibility of the header across a variety of devices and screen sizes. With these improvements, the DfE Frontend framework continues to evolve, ensuring that headers are consistent and functional on all platforms. ## Continued focus on accessibility and consistency DfE Frontend 2.1.0 remains aligned with the departmentâ€™s overarching goals of ensuring accessible and consistent user experiences across all digital services.

Prototyping {% markdown %} # {{ title }} {{ description }} > This guide is a basic walkthrough of setting up and deploying a prototype to GitHub and Heroku. We recommend you follow the GOV.UK service manual [prototyping tutorials and guides](https://prototype-kit.service.gov.uk/docs/tutorials-and-guides) for more detailed guidance on prototyping. This guide will help you to: - set up your device for prototyping using the GOV.UK Prototype kit - create a new prototype - learn about using a terminal and commands - manage source code using GitHub - deploy and secure your prototype to Heroku ## Using terminal commands We're going to use terminal commands throughout this guide to set up your device for prototyping. We'll explain what the commands do and how they work.

When to use the DfE header

If you product or website is hosted on an education.gov.uk domain, you must use the DfE header.

You must also use the DfE header for internal products and websites.

When not to use the DfE header

Do not use the DfE header for products and websites hosted on:

  • gov.uk/[myservice]
  • [myservice].service.gov.uk

{{description}}

When to use the DfE header

If you product or website is hosted on an education.gov.uk domain, you must use the DfE header.

You must also use the DfE header for internal products and websites.

When not to use the DfE header

Do not use the DfE header for products and websites hosted on:

  • gov.uk/[myservice]
  • [myservice].service.gov.uk

Service name

The service name should be implemented within the GOV.UK Service Navigation component. It should not be set in the DfE header.

Navigation

The DfE header component should be used with the GOV.UK Service Navigation component directly adjacent to the header component.

Set the showNav attribute to false to hide the navigation element of the DfE header.

You can follow the example code to implement the DfE header with service name and navigation.

{% markdown %} This is how to best use [the Ministry of Justice’s filter component](https://design-patterns.service.justice.gov.uk/components/filter/) to help users of DfE services refine their search results. ## Guidance You should [follow the DfE Frontend installation instructions](https://design.education.gov.uk/design-system/dfe-frontend/installation) before using this component. ## Adding MoJ components to your prototype You can read the guidance on [how to use the MoJ design system in the GOV.UK prototyping kit](https://design-patterns.service.justice.gov.uk/prototyping/setting-up-coded-prototypes/). If you’re using Figma, there is [an MoJ Figma kit](https://design-patterns.service.justice.gov.uk/prototyping/setting-up-figma-prototypes/) which contains the filter for use in prototypes. ## Use filters on the left first You should place filters on the left of the screen. This allows the list of results to take up the middle and right-hand columns. This can make it easier for users to notice changes to the list of results as they select and apply filters. {% endmarkdown %}

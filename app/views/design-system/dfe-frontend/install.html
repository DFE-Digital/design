This guide is currently written for people wanting to use DfE Frontend in a prototype and using VS Code. If you are looking to use it in .NET apps and unsure how to proceed, contact DesignOps.

Prerequisites

You will need to have the following installed:

  • Node.js
  • VS Code

You will also need to have a prototype set up in VS Code. Follow the instructions on the GOV.UK service manual to get a new prototype up and running.

Install the package

1. Install

Run this command in your terminal in VS Code to install the DfE Frontend package.

npm install dfe-frontend

CSS, JavaScript, and logo files

2. application.scss

Add this to the top of your application.scss file to include the DfE Frontend CSS file, after importing GOV Frontend classes.

@import 'node_modules/dfe-frontend/packages/dfefrontend';

JavaScript

3. layout file

Add this to the bottom of your layout file to include the DfE Frontend JavaScript file.

{%raw%}{% block scripts %}
    {{ super() }}
    <script src="/assets/js/dfefrontend.min.js"></script>
{% endblock %}{%endraw%}

Copy the logo files

4. Copy image assets

Copy these files to your project's assets/images folder. You may need to create this folder if it does not exist.

/node_modules/dfe-frontend/packages/assets/dfe-logo.png
/node_modules/dfe-frontend/packages/assets/dfe-logo-alt.png

Changing the page width

5. Set page width to 1200px

This will set the page width to 1200px. You can adjust this to suit your needs and overrides the GOV.UK smaller 960px width.

.govuk-width-container {
    @include govuk-width-container;
}

Run your app

6. Run your app

In your terminal in VS Code, run this command. It will build and start the app, and you can view it in your browser.

npm run dev

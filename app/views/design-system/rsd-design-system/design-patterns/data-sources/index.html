When to use this pattern

This pattern should be used when:

  • information is automatically taken from an external service or product and displayed on a page
  • Maybe a point here

When not to use this pattern

This pattern should not be used when:

  • information is automatically taken from a different page within the same service or product
  • Maybe a point here

How it works

Generally follows a rule of declaring how much information is being references (e.g. "all" or a specific part), then declaring where it has come from (i.e. external source or internal) and the date of when it was taken.

Follow the same groupings as how the information is being presented on the page

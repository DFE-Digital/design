Design system {% markdown %} # {{ title }} The DfE Header (Rebrand) is a new version of the DfE Header that supports the GOV.UK rebrand. We are moving to use the GOV.UK Header component for all services and replace the GOV.UK logotype with the DfE logo. This guide will help you update your service to use the new DfE Header (Rebrand). ## Before you start The new GOV.UK Header does not include options for: * service name - this is now handled by the GOV.UK [Service Navigation](https://design-system.service.gov.uk/components/service-navigation/) component * service navigation - this is now handled by service navigation component * search - we are waiting for the GOV.UK Header to include how they want us to implement search * sign in and sign out links - we are waiting for the GOV.UK Header to include how they want us to implement sign in and sign out links ## 1. Update your GOV.UK Frontend version Update your code to use GOV.UK Frontend 5.10.0 or above. You can [read the release notes](https://github.com/alphagov/govuk-frontend/releases/tag/v5.10.0) for more information about the changes and how to implement them. This will depend on your codebase, for example, Ruby, Node, .NET. You should know how to update your codebase, so we will not cover this in this guide. ## 2. Update your layout to use the reband styles Add the govuk-template--rebranded class to the <html> tag in your layout file: ```html ``` ## 3. Update your logo > We are moving to use the "Lesser Arms Landscape" logo instead of the "Lesser Arms Stacked" logo. You need to download and include the new landscape logos in the folder where you serve image assets for your product. This could be in a folder called public/assets/images or public/images or similar. | Logo | Description | |------|-------------| | [department-for-education_white.png](/assets/images/rebrand/department-for-education_white.png) | The white logo for use on the blue background | | [department-for-education_black.png](/assets/images/rebrand/department-for-education_black.png) | The black logo which is used when the logo is in focus state | Do not under any circumstances alter the size, resolution or aspect ratio of the logos. The landscape logo is artwork provided by [UK Government Brand centre](https://hmgbrand.gcs.civilservice.gov.uk/login/?mepr-unauth-page=1641&redirect_to=%2Fassets-and-artwork%2F). ## 4. Update your CSS Add the following CSS to your stylesheet after the existing styles. This will ensure the logo is the correct size and position and that the focus state is applied correctly. ```css .govuk-header__link .govuk-header__logotype{ height: 30px; } .govuk-header__link:focus .govuk-header__logotype{ content: url(/public/images/department-for-education_black.png); height: 30px; } ``` ## 5. Use and update the GOV.UK Header component For all services, use the GOV.UK Header component as the header for your service.

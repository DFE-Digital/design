Always use the GOV.UK colour palette first.

Where you want to use Department for Education specific branding colours, you can use the following colour palette.

Colour contrast

You must make sure that the contrast ratio of text and interactive elements in
your service meets 1.4.3 - Contrast - level AA of the Web Content Accessibility Guidelines (WCAG 2.1).

Do not use the hexadecimal (hex) codes directly, use the mixin name. For example, if you wanted to use the DfE department colour blue use: $color_dfe-blue instead of #003a69.

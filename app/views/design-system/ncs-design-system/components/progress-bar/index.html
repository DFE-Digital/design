Progress bar

The progress bar is a visual status representation of how a user is progressing in the DYSAC asscessment.

It helps set expectations, provides instant feedback and reduces user friction.

{{ sample.code('/design-system/ncs-design-system/components/progress-bar/example.html', 'Progress bar example', 'progress-bar', htmlCode) }}

When to use this component

You can use a progress bar on forms and assessments on the National Careers Service website.

This sets expectations for users, allowing them to understand where they are in the journey and how much longer they have until they've completed the task.

When not to use this component

You do not need a progress bar on forms that are simple and short.

Only add a progress bar if you can do so reliably with the number shown increasing as the user moves through the form.

How to use this component

There are 2 ways to use the error summary component. You can use HTML or, if you're using Nunjucks or the GOV.UK Prototype Kit, you can use the Nunjucks macro.

Header

Top Header

The top header shows users that they are on National Carrer Service .

{{ sample.code('/design-system/ncs-design-system/components/header/top-header-example.html', 'Top header example', 'top-header', htmlCode_header) }}

Homepage header

The homepage header should be used only at the top of the homepage. Images should be used with thought as to how the text and white text background covers the image.

Headers are all full width. The National Careers Service logo is only required on desktop views

H1 text is required but other text is optional.

{{ sample.code('/design-system/ncs-design-system/components/header/homepage-header-example.html', 'Homepage header example', 'homepage-header-box', htmlCode_homepage) }}

Page header

Unlike the homepage header, the page header can have multiple text elements. H1 text is required but other text is optional. Text should be confined to a two third column. Headers are all full width.

{{ sample.code('/design-system/ncs-design-system/components/header/page-header-example.html', 'Page header example', 'page-header-box', htmlCode_page) }}

When to use this component

You must use the National Careers Service header at the top of every page if your service is being hosted on the nationalcareers.service.gov.uk domain.

The homepage header should be used only at the top of the homepage. Images should be used with thought as to how the text and white text background covers the image.

A page header should be used on pages other than the homepage and only at the top. Images should be used with thought as to how the text and white text background covers the image.

When not to use this component

You must not use the National Careers Service header if your service is not hosted on the National Careers Service domain.

How to use this component

If you use the page template, you'll get the header without having to add it as it's included by default. However, if you need to add this without using the page template, you can copy the HTML and paste it where you'd like to display the header on your page.

Once you've added the code, you can edit the navigation text and links to suit your needs.
